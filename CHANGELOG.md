# 수정 및 추가 사항 정리

## 1. 모델 사이즈 및 추천 섹션
- **레이아웃**: 모델 이미지 50% / 텍스트 50% 비율로 통일
- 각 행: 왼쪽 이미지, 오른쪽 인치·추천 공간·추천 인원
- 모바일: 이미지 → 텍스트 순 세로 배치 유지

---

## 2. 이미지 & 시연 섹션

### 2-1. 구조·디자인
- **이미지 / 영상 구분**: 이미지 블록과 시연 영상 블록을 분리 배치
  - **이미지**: Eshare(풀와이드) + UMIND · 교실 수업(2열)
  - **시연 영상**: 유튜브 4편을 2열 그리드로 배치
- 카드 스타일 통일: 상단 시각(이미지/영상) + 하단 텍스트 블록, `border-t`로 구분
- 서브라벨 + "→" (예: "Eshare · 3초 연결 →", "시연 영상 보기 →") 적용

### 2-2. Eshare 카드
- **배경**: 이미지의 하얀 배경에 맞춰 카드·이미지 영역 배경을 흰색(`bg-white`)으로 변경
- 하단 텍스트 영역은 기존 다크(`bg-navyLight`) 유지

### 2-3. 시연 영상
- **영상 1개 추가**: YouTube Shorts (`Ofl5GWPY2lQ`) — "넥소 시연 쇼츠"
- **영상 카드 확대**: 3열 → 2열로 변경해 카드당 크기 확대
- 영상 영역 `min-h-[220px]`(모바일), `md:min-h-[320px]`(데스크톱) 적용으로 쇼츠가 더 크게 보이도록 조정

---

## 3. 전체 타이포그래피 (시원한 느낌)

### 3-1. 전역 (`index.css`)
- `body`: 기본 글자 크기 **17px**(1.0625rem), **줄간격 1.75**, **자간 0.01em**

### 3-2. Tailwind (`tailwind.config.js`)
- `fontSize`: 본문용 `body`, 강조용 `lead` 추가
- `lineHeight`: `relaxed`(1.75), `loose`(2)
- `letterSpacing`: `comfort`(0.02em)

### 3-3. 섹션별 적용
- **제목**: `leading-snug`, 필요 시 `tracking-tight`
- **본문·설명**: `text-base` + `leading-relaxed` (기존 `text-sm` 확대)
- **히어로**: 서브 카피 `leading-relaxed`, 제목 여백 확대
- **카드 설명·모델 추천·이미지&시연 텍스트**: 위 기준으로 통일
- **루트**: `antialiased` 적용
